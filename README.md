# Coloured_Ball_Catch_Game
Coloured Ball Catch Game

Introduction:
=============
This game is Python-tkinter based. As "tkinter" is inbuilt package coming with python installer,
no need to install any additional thing. Simply import works well inside the code.
In order to play we can use left-right arrow keys or simply mouse movement in x-direction.
If we eat green balls, our score keep on increasing. We get 3 lives in this game.
By mistake if we eat a red ball one life point is gone and the Lives point reduces.
Once the lives point reduces to zero, the game is over. That time you can note your maximum score.
